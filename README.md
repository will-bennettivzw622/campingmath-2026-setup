## What CampingMath Does

CampingMath packages several outdoor planning calculators into a single static web app. Rather than switching between notes, rough estimates, and separate tools, it gives you a quick way to answer common questions that come up while preparing a camping or hiking trip.

The app is aimed at users who need straightforward calculations for gear, campsite setup, food and fuel estimates, and hiking pace. Since it is built as a static site with HTML and JavaScript, it is simple to host, fast to open, and usable in any browser without extra runtime requirements.
